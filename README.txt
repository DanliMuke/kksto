Бот для пересылки сообщений из лички в группу

Файлы:
- bot.py - сам бот
- requirements.txt - Python-зависимости
- .env.example - пример нужных переменных окружения
- install_requirements.bat - установка зависимостей
- start_bot.bat - запуск бота

Перед запуском нужно задать переменные окружения:
BOT_TOKEN
GROQ_API_KEY
TARGET_CHAT_ID

Важно: не публикуй токен бота и ключ Groq. Если токен уже попадал в чужие руки или публичный лог, лучше перевыпустить его через BotFather.
